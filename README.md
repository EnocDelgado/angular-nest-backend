# Nest Backend

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Pre Installation

If node_modules was not created, run the following comand:

```bash
$ npm install --silent
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Start DB on Docker

 After configure our docker-compose.yml run following comand:

 ```
$ docker compose up -d
 ```
